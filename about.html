About PlayersGames

PlayersGames is a simple Bootstrap-based site to showcase player and game data. It’s intentionally lightweight so it can be developed, versioned in GitHub, and deployed quickly to the web.

Tech stack

  • Frontend: HTML5, CSS, Bootstrap 5
  • Source Control: Git & GitHub
  • Deployment: GitHub Pages + Custom Domain
  • Editor: Visual Studio Code

Concept

The idea is to treat this project as a starting point: you can extend the Data page to hook into a real database or API later (for example using Python/Flask, Node, or any backend of your choice).

For now, it’s a static, fast-loading site that you can use as a personal or demo project, hosted under your own domain.
